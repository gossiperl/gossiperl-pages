language agnostic, Apache Thrift based gossip
middleware and message bus

pronounced [ `gossip,erl ]

Gossiperl is a language agnostic, Apache Thrift based gossip middleware and message bus for data dissemination and replication within global cloud environments. Gossiperl provides gossip protocol as a service. Gossiperl is:

  • highly scalable: written in Erlang OTP
  • highly secure: all traffic encrypted
  • flexible: no defined communication patterns, applications can form communication in any way desired for the implementation
  • content independent: any kind data can be distributed, Apache Thrift
  • configurable: managed with REST API, run-time reconfiguration
